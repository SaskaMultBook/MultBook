Эталонный финальный архив MultBook.

Структура:
- 01_text — тексты (якутский и русский)
- 02_scenes — сцены (JSON или описания)
- 03_prompts — промты для изображений/видео
- 04_images — картинки для сцен
- 05_audio — дополнительные аудио
- 05_audio_prompts — английские промты для озвучки
- 06_audio_voice — записи и озвучки
- 06_video — видеофайлы
- MultBook_backup — бэкапы
- Tools — вспомогательные скрипты/инструменты

Файлы:
- negative_prompt.txt — единый негативный промт
- push_to_github.bat — пуш в GitHub
- backup_multbook.bat — создание бэкапа
- .gitignore — игнорирование лишних файлов

Все батники оставляют окно открытым.
